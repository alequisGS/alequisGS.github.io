# Alex Gomez Academic Website

This repository contains the source for Alex Gomez's personal academic website,
hosted with GitHub Pages at:

https://alequisGS.github.io

The site is intentionally simple: plain HTML and CSS, with no framework or build
step. The main page is `index.html`, styling is in `style.css`, and static
assets such as the profile photo, CV, and thesis-project PDF live in `assets/`.

To preview locally, open `index.html` in a web browser.
